Arduino Car with obstacle avoidance and line following capabilities. It can also automatically switch between the 2 modes.
It used ultrasonic sensors for obstacle avoidance and uses IR sensors for following the line. Includes a 4-wheel setup with individual control, allowing it to turn smoothly. It is battery-powered and operated by Arduino.
